SellerCentral Chatbot

RAG LangChain FAISS Hugging Face LLaMA BigQuery Docker FastAPI Airflow GCP

SellerCentral Chatbot is a production-grade RAG (retrieval-augmented generation) assistant that lets Amazon sellers ask natural-language questions about their product metrics, reviews, and store performance.

  • Designed and deployed a production RAG pipeline using LangChain and LLaMA (via Groq); ran systematic experiments across Hugging Face embedding models, FAISS indexing strategies, and chunking configurations. Benchmarked against baseline LLM approaches using Precision@k, Recall@k, BLEU, and ROUGE-L — achieving 95% response accuracy and sub-2-second latency for 90% of queries.
  • Established a data-quality framework with automated anomaly detection and bias checks across hundreds of thousands of records (flagging 27% as anomalous), enforced by 7 test modules covering schema validation, DAG execution, and ETL correctness.
  • Architected an Airflow ETL pipeline ingesting Amazon review data into BigQuery with incremental loading across 4 product categories; deployed end-to-end as a Dockerized FastAPI service on GCP Cloud Run with CI/CD via GitHub Actions.
