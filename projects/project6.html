AI Teaching Assistant

RAG FastAPI React pgvector LLaMA (Groq) OpenAI Embeddings scikit-learn GCP Cloud Run PostgreSQL

AI Teaching Assistant is an AI-powered platform for university database-systems courses that helps faculty run learning analytics, generate content, and give students personalized support — all through a React dashboard backed by a FastAPI service on GCP. Built as my CS 5500 Data Science Capstone at Northeastern.

What I built

  • RAG-based Q&A: answers course questions grounded in uploaded lecture PDFs with source citations — documents are auto-chunked and embedded (OpenAI ada-002) into PostgreSQL + pgvector on upload, with generation via LLaMA 3.1 (Groq).
  • At-risk student detection: a scikit-learn model flags struggling students from grades, attendance, and quiz performance, surfaced in a learning-analytics dashboard alongside grade distributions and completion trends.
  • Content generation: AI quiz generation (multiple question types with distractor explanations), 16-week course planning, auto-generated presentation slides (python-pptx / reportlab), and an AI-assisted rubric builder.
  • Student hub: a separate interface where students view grades, take quizzes, and participate in the RAG-powered Q&A.
  • Deployed on GCP — Cloud Run, Cloud Storage, and Cloud Functions (PDF ingestion on upload) — with a React 18 / Vite / Tailwind frontend served via nginx over the FastAPI backend.
